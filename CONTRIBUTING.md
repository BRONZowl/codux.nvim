# Contributing

Thanks for your interest in contributing to codux.nvim.

## Reporting Issues

Please include:
- Neovim version
- Plugin version
- Operating system
- Steps to reproduce

## Pull Requests

- Keep changes focused
- Update documentation when needed
- Test before submitting
